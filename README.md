# AssetFlow (Odoo)

AssetFlow is a comprehensive asset management and allocation system.

## 🚀 Live Demo

**[View the Live Application on GitHub Pages](https://mayankidk.github.io/Odoo/)**

## 🛠️ Tech Stack

- **Frontend:** React, TypeScript, Vite, Tailwind CSS, Lucide Icons, TanStack Query
- **Backend:** Supabase (PostgreSQL, Auth, Storage)

## 📦 Getting Started

### Prerequisites

- Node.js
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Mayankidk/Odoo.git
   ```

2. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```
